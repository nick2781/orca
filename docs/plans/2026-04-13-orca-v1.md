# Orca V1 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build orca — a daemon-based multi-agent orchestrator where Claude Code acts as brain and Codex workers execute implementation tasks, with 3-level escalation, smart worktree isolation, and native terminal integration.

**Architecture:** Single Rust binary (`orca`) serves as daemon (`orca daemon start`), CLI client, and MCP server (`orca mcp-server`). Daemon manages task lifecycle via Unix socket JSON-RPC. Workers are spawned as child processes with structured output parsing.

**Tech Stack:** Rust, tokio, clap, serde, rmcp (MCP SDK), git2, tracing, ratatui (status bar)

```rust
use chrono::{DateTime, Utc};
use serde::{Deserialize, Serialize};
use uuid::Uuid;

/// Task specification — what CC sends to the daemon
#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct TaskSpec {
    pub id: String,
    pub title: String,
    pub description: String,
    pub context: TaskContext,
    #[serde(default)]
    pub isolation: IsolationMode,
    #[serde(default)]
    pub depends_on: Vec<String>,
    #[serde(default = "default_priority")]
    pub priority: u32,
}

fn default_priority() -> u32 {
    1
}

#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct TaskContext {
    #[serde(default)]
    pub files: Vec<String>,
    #[serde(default)]
    pub references: Vec<String>,
    #[serde(default)]
    pub constraints: String,
}

#[derive(Debug, Clone, Serialize, Deserialize, Default, PartialEq, Eq)]
#[serde(rename_all = "snake_case")]
pub enum IsolationMode {
    #[default]
    Auto,
    Worktree,
    Serial,
}

#[derive(Debug, Clone, Serialize, Deserialize, PartialEq, Eq)]
#[serde(rename_all = "snake_case")]
pub enum TaskState {
    Pending,
    Assigned,
    Running,
    Done,
    Blocked,
    Review,
    Accepted,
    Rejected,
    Completed,
    Cancelled,
}

/// Runtime task — TaskSpec + runtime state
#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct Task {
    pub spec: TaskSpec,
    pub state: TaskState,
    pub worker_id: Option<String>,
    pub escalation_id: Option<String>,
    pub output: Option<TaskOutput>,
    pub created_at: DateTime<Utc>,
    pub updated_at: DateTime<Utc>,
    pub worktree_path: Option<String>,
    pub branch_name: Option<String>,
}

#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct TaskOutput {
    pub files_changed: Vec<String>,
    pub tests_passed: Option<bool>,
    pub diff_summary: String,
    pub stdout: String,
}

/// Plan — what CC submits via orca_plan
#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct Plan {
    pub id: String,
    pub tasks: Vec<TaskSpec>,
    pub dependencies: Vec<Edge>,
    pub created_at: DateTime<Utc>,
}

#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct Edge {
    pub from: String,
    pub to: String,
}

/// Worker status
#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct WorkerInfo {
    pub id: String,
    pub worker_type: String,
    pub status: WorkerStatus,
    pub current_task_id: Option<String>,
    pub started_at: DateTime<Utc>,
}

#[derive(Debug, Clone, Serialize, Deserialize, PartialEq, Eq)]
#[serde(rename_all = "snake_case")]
pub enum WorkerStatus {
    Idle,
    Busy,
    Dead,
}

impl Task {
    pub fn new(spec: TaskSpec) -> Self {
        let now = Utc::now();
        Self {
            spec,
            state: TaskState::Pending,
            worker_id: None,
            escalation_id: None,
            output: None,
            created_at: now,
            updated_at: now,
            worktree_path: None,
            branch_name: None,
        }
    }

    /// Returns true if the state transition is valid
    pub fn can_transition_to(&self, target: &TaskState) -> bool {
        use TaskState::*;
        matches!(
            (&self.state, target),
            (Pending, Assigned)
                | (Assigned, Running)
                | (Assigned, Cancelled)
                | (Running, Done)
                | (Running, Blocked)
                | (Running, Cancelled)
                | (Blocked, Running) // resumed
                | (Done, Review)
                | (Review, Accepted)
                | (Review, Rejected)
                | (Accepted, Completed)
                | (Rejected, Pending) // rework
        )
    }

    pub fn transition_to(&mut self, target: TaskState) -> anyhow::Result<()> {
        if !self.can_transition_to(&target) {
            anyhow::bail!(
                "invalid state transition: {:?} -> {:?} for task {}",
                self.state,
                target,
                self.spec.id
            );
        }
        self.state = target;
        self.updated_at = Utc::now();
        Ok(())
    }
}
```

```rust
use serde::{Deserialize, Serialize};
use serde_json::Value;

/// JSON-RPC 2.0 request
#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct RpcRequest {
    pub jsonrpc: String,
    pub id: Option<Value>,
    pub method: String,
    #[serde(default)]
    pub params: Value,
}

/// JSON-RPC 2.0 response
#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct RpcResponse {
    pub jsonrpc: String,
    pub id: Option<Value>,
    #[serde(skip_serializing_if = "Option::is_none")]
    pub result: Option<Value>,
    #[serde(skip_serializing_if = "Option::is_none")]
    pub error: Option<RpcError>,
}

#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct RpcError {
    pub code: i32,
    pub message: String,
    #[serde(skip_serializing_if = "Option::is_none")]
    pub data: Option<Value>,
}

// Standard JSON-RPC error codes
pub const PARSE_ERROR: i32 = -32700;
pub const INVALID_REQUEST: i32 = -32600;
pub const METHOD_NOT_FOUND: i32 = -32601;
pub const INVALID_PARAMS: i32 = -32602;
pub const INTERNAL_ERROR: i32 = -32603;

// Application error codes
pub const TASK_NOT_FOUND: i32 = -32000;
pub const INVALID_STATE_TRANSITION: i32 = -32001;
pub const WORKER_NOT_FOUND: i32 = -32002;
pub const ESCALATION_NOT_FOUND: i32 = -32003;

impl RpcRequest {
    pub fn new(method: &str, params: Value) -> Self {
        Self {
            jsonrpc: "2.0".into(),
            id: Some(Value::String(uuid::Uuid::new_v4().to_string())),
            method: method.into(),
            params,
        }
    }
}

impl RpcResponse {
    pub fn success(id: Option<Value>, result: Value) -> Self {
        Self {
            jsonrpc: "2.0".into(),
            id,
            result: Some(result),
            error: None,
        }
    }

    pub fn error(id: Option<Value>, code: i32, message: &str) -> Self {
        Self {
            jsonrpc: "2.0".into(),
            id,
            result: None,
            error: Some(RpcError {
                code,
                message: message.into(),
                data: None,
            }),
        }
    }
}
```

```rust
// src/daemon/server.rs
use crate::protocol::{RpcRequest, RpcResponse, METHOD_NOT_FOUND};
use anyhow::Result;
use std::path::Path;
use std::sync::Arc;
use tokio::io::{AsyncBufReadExt, AsyncWriteExt, BufReader};
use tokio::net::UnixListener;
use tokio::sync::Mutex;
use tracing::{info, warn, error};

pub type RpcHandler = Arc<dyn Fn(RpcRequest) -> RpcResponse + Send + Sync>;

pub struct IpcServer {
    listener: UnixListener,
    handler: RpcHandler,
}

impl IpcServer {
    pub async fn bind(socket_path: &Path, handler: RpcHandler) -> Result<Self> {
        // Remove stale socket file
        if socket_path.exists() {
            std::fs::remove_file(socket_path)?;
        }
        if let Some(parent) = socket_path.parent() {
            std::fs::create_dir_all(parent)?;
        }

        let listener = UnixListener::bind(socket_path)?;
        info!("IPC server listening on {:?}", socket_path);

        Ok(Self { listener, handler })
    }

    pub async fn run(self) -> Result<()> {
        loop {
            let (stream, _) = self.listener.accept().await?;
            let handler = self.handler.clone();

            tokio::spawn(async move {
                let (reader, mut writer) = stream.into_split();
                let mut reader = BufReader::new(reader);
                let mut line = String::new();

                loop {
                    line.clear();
                    match reader.read_line(&mut line).await {
                        Ok(0) => break, // EOF
                        Ok(_) => {
                            let trimmed = line.trim();
                            if trimmed.is_empty() {
                                continue;
                            }

                            let response = match serde_json::from_str::<RpcRequest>(trimmed) {
                                Ok(req) => (handler)(req),
                                Err(e) => RpcResponse::error(
                                    None,
                                    crate::protocol::PARSE_ERROR,
                                    &format!("parse error: {}", e),
                                ),
                            };

                            let mut resp_bytes =
                                serde_json::to_vec(&response).unwrap_or_default();
                            resp_bytes.push(b'\n');
                            if writer.write_all(&resp_bytes).await.is_err() {
                                break;
                            }
                        }
                        Err(e) => {
                            warn!("read error: {}", e);
                            break;
                        }
                    }
                }
            });
        }
    }
}

/// Client for connecting to the daemon IPC
pub struct IpcClient {
    stream: tokio::net::UnixStream,
}

impl IpcClient {
    pub async fn connect(socket_path: &Path) -> Result<Self> {
        let stream = tokio::net::UnixStream::connect(socket_path).await?;
        Ok(Self { stream })
    }

    pub async fn call(&mut self, request: &RpcRequest) -> Result<RpcResponse> {
        let (reader, mut writer) = self.stream.split();
        let mut reader = BufReader::new(reader);

        let mut req_bytes = serde_json::to_vec(request)?;
        req_bytes.push(b'\n');
        writer.write_all(&req_bytes).await?;

        let mut line = String::new();
        reader.read_line(&mut line).await?;
        let response: RpcResponse = serde_json::from_str(line.trim())?;
        Ok(response)
    }
}
```

```rust
// src/daemon/scheduler.rs
use crate::types::*;
use std::collections::{HashMap, HashSet, VecDeque};
use anyhow::Result;

/// Dependency graph — determines which tasks can run concurrently
pub struct DependencyGraph {
    /// task_id -> set of task_ids it depends on
    deps: HashMap<String, HashSet<String>>,
    /// task_id -> set of task_ids that depend on it
    reverse_deps: HashMap<String, HashSet<String>>,
}

impl DependencyGraph {
    pub fn new(tasks: &[TaskSpec], edges: &[Edge]) -> Result<Self> {
        let task_ids: HashSet<String> = tasks.iter().map(|t| t.id.clone()).collect();
        let mut deps: HashMap<String, HashSet<String>> = HashMap::new();
        let mut reverse_deps: HashMap<String, HashSet<String>> = HashMap::new();

        for task in tasks {
            deps.entry(task.id.clone()).or_default();
            reverse_deps.entry(task.id.clone()).or_default();
        }

        for edge in edges {
            if !task_ids.contains(&edge.from) || !task_ids.contains(&edge.to) {
                anyhow::bail!(
                    "edge references unknown task: {} -> {}",
                    edge.from,
                    edge.to
                );
            }
            deps.entry(edge.to.clone())
                .or_default()
                .insert(edge.from.clone());
            reverse_deps
                .entry(edge.from.clone())
                .or_default()
                .insert(edge.to.clone());
        }

        // Cycle detection via topological sort
        let graph = Self { deps, reverse_deps };
        graph.validate_no_cycles(&task_ids)?;

        Ok(graph)
    }

    fn validate_no_cycles(&self, task_ids: &HashSet<String>) -> Result<()> {
        let mut in_degree: HashMap<&str, usize> = HashMap::new();
        for id in task_ids {
            in_degree.insert(id.as_str(), self.deps.get(id).map_or(0, |d| d.len()));
        }

        let mut queue: VecDeque<&str> = in_degree
            .iter()
            .filter(|(_, &deg)| deg == 0)
            .map(|(&id, _)| id)
            .collect();

        let mut visited = 0;
        while let Some(id) = queue.pop_front() {
            visited += 1;
            if let Some(dependents) = self.reverse_deps.get(id) {
                for dep in dependents {
                    if let Some(deg) = in_degree.get_mut(dep.as_str()) {
                        *deg -= 1;
                        if *deg == 0 {
                            queue.push_back(dep.as_str());
                        }
                    }
                }
            }
        }

        if visited != task_ids.len() {
            anyhow::bail!("dependency cycle detected");
        }
        Ok(())
    }

    /// Tasks that are ready to run (all dependencies completed)
    pub fn ready_tasks(&self, completed: &HashSet<String>) -> Vec<String> {
        self.deps
            .iter()
            .filter(|(id, dep_set)| {
                !completed.contains(*id) && dep_set.iter().all(|d| completed.contains(d))
            })
            .map(|(id, _)| id.clone())
            .collect()
    }

    pub fn dependencies_of(&self, task_id: &str) -> &HashSet<String> {
        static EMPTY: HashSet<String> = HashSet::new();
        self.deps.get(task_id).unwrap_or(&EMPTY)
    }
}

/// Scheduler — assigns tasks to workers
pub struct Scheduler {
    graph: DependencyGraph,
}

impl Scheduler {
    pub fn new(graph: DependencyGraph) -> Self {
        Self { graph }
    }

    /// Determine which tasks can be assigned given current state
    pub fn assignable_tasks(
        &self,
        tasks: &HashMap<String, Task>,
        max_workers: usize,
        active_count: usize,
    ) -> Vec<String> {
        let completed: HashSet<String> = tasks
            .iter()
            .filter(|(_, t)| matches!(t.state, TaskState::Completed | TaskState::Accepted))
            .map(|(id, _)| id.clone())
            .collect();

        let in_progress: HashSet<String> = tasks
            .iter()
            .filter(|(_, t)| {
                matches!(
                    t.state,
                    TaskState::Assigned | TaskState::Running | TaskState::Review
                )
            })
            .map(|(id, _)| id.clone())
            .collect();

        let available_slots = max_workers.saturating_sub(active_count);
        if available_slots == 0 {
            return vec![];
        }

        self.graph
            .ready_tasks(&completed)
            .into_iter()
            .filter(|id| {
                tasks
                    .get(id)
                    .map_or(false, |t| t.state == TaskState::Pending)
            })
            .filter(|id| !in_progress.contains(id))
            .take(available_slots)
            .collect()
    }

    /// Check if two tasks have file overlap (for isolation decision)
```

```rust
const MARKER_DONE: &str = "[ORCA:DONE]";
const MARKER_ESCALATE: &str = "[ORCA:ESCALATE]";
const MARKER_BLOCKED: &str = "[ORCA:BLOCKED]";
const MARKER_PROGRESS: &str = "[ORCA:PROGRESS]";

/// Parse a single line of worker output into a WorkerMessage
pub fn parse_worker_line(line: &str) -> WorkerMessage {
    let trimmed = line.trim();

    if let Some(rest) = trimmed.strip_prefix(MARKER_DONE) {
        let rest = rest.trim();
        match serde_json::from_str::<TaskOutput>(rest) {
            Ok(output) => WorkerMessage::Done(output),
            Err(_) => WorkerMessage::Done(TaskOutput {
                files_changed: vec![],
                tests_passed: None,
                diff_summary: String::new(),
                stdout: rest.to_string(),
            }),
        }
    } else if let Some(rest) = trimmed.strip_prefix(MARKER_ESCALATE) {
        let json = serde_json::from_str(rest.trim()).unwrap_or(serde_json::json!({"raw": rest.trim()}));
        WorkerMessage::Escalate(json)
    } else if let Some(rest) = trimmed.strip_prefix(MARKER_BLOCKED) {
        let json = serde_json::from_str(rest.trim()).unwrap_or(serde_json::json!({"raw": rest.trim()}));
        WorkerMessage::Blocked(json)
    } else if let Some(rest) = trimmed.strip_prefix(MARKER_PROGRESS) {
        WorkerMessage::Progress(rest.trim().to_string())
    } else {
        WorkerMessage::Output(line.to_string())
    }
}

/// Generate prompt for Codex from a TaskSpec
pub fn generate_prompt(task: &TaskSpec, work_dir: &str) -> String {
    let mut prompt = format!("You are working on task: {}\n\n", task.title);
    prompt.push_str(&format!("## Description\n{}\n\n", task.description));

    if !task.context.files.is_empty() {
        prompt.push_str("## Files to modify\n");
        for f in &task.context.files {
            prompt.push_str(&format!("- {}\n", f));
        }
        prompt.push('\n');
    }

    if !task.context.constraints.is_empty() {
        prompt.push_str(&format!("## Constraints\n{}\n\n", task.context.constraints));
    }

    if !task.context.references.is_empty() {
        prompt.push_str("## References\n");
        for r in &task.context.references {
            prompt.push_str(&format!("- {}\n", r));
        }
        prompt.push('\n');
    }

    prompt.push_str(&format!("## Working directory\n{}\n\n", work_dir));

    prompt.push_str("## Rules\n");
    prompt.push_str("- When you need a decision, output: [ORCA:ESCALATE] {json}\n");
    prompt.push_str("- When done, output: [ORCA:DONE] {json summary}\n");
    prompt.push_str("- When you hit a blocker, output: [ORCA:BLOCKED] {json reason}\n");
    prompt.push_str("- Report progress with: [ORCA:PROGRESS] <message>\n");
    prompt.push_str("- Do not modify files outside the specified scope\n");

    prompt
}

pub struct CodexWorker {
    config: WorkerConfig,
    processes: Arc<Mutex<HashMap<String, Child>>>,
}

impl CodexWorker {
    pub fn new(config: WorkerConfig) -> Self {
        Self {
            config,
            processes: Arc::new(Mutex::new(HashMap::new())),
        }
    }
}

#[async_trait]
impl Worker for CodexWorker {
    async fn spawn(&self, worker_id: &str, work_dir: &str) -> Result<()> {
        let child = Command::new(&self.config.command)
            .args(&self.config.args)
            .current_dir(work_dir)
            .stdin(std::process::Stdio::piped())
            .stdout(std::process::Stdio::piped())
            .stderr(std::process::Stdio::piped())
            .spawn()?;

        self.processes
            .lock()
            .await
            .insert(worker_id.to_string(), child);
        info!("spawned codex worker {} in {}", worker_id, work_dir);
        Ok(())
    }

    async fn dispatch(&self, worker_id: &str, task: &TaskSpec) -> Result<()> {
        let mut procs = self.processes.lock().await;
        let child = procs
            .get_mut(worker_id)
            .ok_or_else(|| anyhow::anyhow!("worker {} not found", worker_id))?;

        let prompt = generate_prompt(task, ".");
        if let Some(stdin) = child.stdin.as_mut() {
            use tokio::io::AsyncWriteExt;
            stdin.write_all(prompt.as_bytes()).await?;
            stdin.write_all(b"\n").await?;
        }
        Ok(())
    }

    async fn health_check(&self, worker_id: &str) -> Result<WorkerStatus> {
        let mut procs = self.processes.lock().await;
        match procs.get_mut(worker_id) {
            Some(child) => match child.try_wait()? {
                Some(_) => Ok(WorkerStatus::Dead),
                None => Ok(WorkerStatus::Busy),
            },
            None => Ok(WorkerStatus::Dead),
        }
    }

    async fn interrupt(&self, worker_id: &str) -> Result<()> {
        let mut procs = self.processes.lock().await;
        if let Some(child) = procs.get_mut(worker_id) {
            child.kill().await?;
        }
        Ok(())
    }

    async fn cleanup(&self, worker_id: &str) -> Result<()> {
        let mut procs = self.processes.lock().await;
        if let Some(mut child) = procs.remove(worker_id) {
            let _ = child.kill().await;
        }
        Ok(())
    }

    fn worker_type(&self) -> &str {
        "codex"
    }
}
```

```rust
// src/isolation.rs
use anyhow::Result;
use std::collections::HashSet;
use std::path::{Path, PathBuf};
use std::process::Command;
use tracing::info;

use crate::types::{IsolationMode, TaskSpec};

pub enum IsolationDecision {
    Worktree { path: PathBuf, branch: String },
    SameDir,
    Serial { wait_for: String },
    AskCc,
}

pub struct IsolationManager {
    worktree_base: PathBuf,
    project_dir: PathBuf,
}

impl IsolationManager {
    pub fn new(project_dir: &Path, worktree_base: &Path) -> Self {
        Self {
            worktree_base: worktree_base.to_path_buf(),
            project_dir: project_dir.to_path_buf(),
        }
    }

    /// Decide isolation strategy for a task
    pub fn decide(
        &self,
        task: &TaskSpec,
        running_tasks: &[&TaskSpec],
    ) -> IsolationDecision {
        match task.isolation {
            IsolationMode::Worktree => {
                let path = self.worktree_base.join(&task.id);
                let branch = format!("orca/{}", task.id);
                IsolationDecision::Worktree { path, branch }
            }
            IsolationMode::Serial => {
                // Find overlapping running task
                for running in running_tasks {
                    if has_file_overlap(task, running) {
                        return IsolationDecision::Serial {
                            wait_for: running.id.clone(),
                        };
                    }
                }
                IsolationDecision::SameDir
            }
            IsolationMode::Auto => {
                if task.context.files.is_empty() {
                    return IsolationDecision::AskCc;
                }

                for running in running_tasks {
                    if has_file_overlap(task, running) {
                        return IsolationDecision::Serial {
                            wait_for: running.id.clone(),
                        };
                    }
                }

                let path = self.worktree_base.join(&task.id);
                let branch = format!("orca/{}", task.id);
                IsolationDecision::Worktree { path, branch }
            }
        }
    }

    /// Create a git worktree for a task
    pub fn create_worktree(&self, path: &Path, branch: &str) -> Result<()> {
        std::fs::create_dir_all(path.parent().unwrap_or(path))?;

        let output = Command::new("git")
            .args(["worktree", "add", "-b", branch])
            .arg(path)
            .current_dir(&self.project_dir)
            .output()?;

        if !output.status.success() {
            let stderr = String::from_utf8_lossy(&output.stderr);
            anyhow::bail!("git worktree add failed: {}", stderr);
        }

        info!("created worktree at {:?} on branch {}", path, branch);
        Ok(())
    }

    /// Remove a git worktree
    pub fn remove_worktree(&self, path: &Path) -> Result<()> {
        let output = Command::new("git")
            .args(["worktree", "remove", "--force"])
            .arg(path)
            .current_dir(&self.project_dir)
            .output()?;

        if !output.status.success() {
            let stderr = String::from_utf8_lossy(&output.stderr);
            anyhow::bail!("git worktree remove failed: {}", stderr);
        }

        info!("removed worktree at {:?}", path);
        Ok(())
    }

    /// Merge a worktree branch into target branch
    pub fn merge_branch(&self, branch: &str, target: &str) -> Result<MergeResult> {
        // Checkout target
        let output = Command::new("git")
            .args(["checkout", target])
            .current_dir(&self.project_dir)
            .output()?;

        if !output.status.success() {
            anyhow::bail!("checkout {} failed", target);
        }

        // Merge
        let output = Command::new("git")
            .args(["merge", branch, "--no-ff", "-m"])
            .arg(format!("merge orca branch: {}", branch))
            .current_dir(&self.project_dir)
            .output()?;

        if output.status.success() {
            Ok(MergeResult::Success)
        } else {
            let stderr = String::from_utf8_lossy(&output.stderr);
            if stderr.contains("CONFLICT") {
                Ok(MergeResult::Conflict(stderr.to_string()))
            } else {
                anyhow::bail!("merge failed: {}", stderr);
            }
        }
    }

    /// Delete a branch after successful merge
    pub fn delete_branch(&self, branch: &str) -> Result<()> {
        Command::new("git")
            .args(["branch", "-d", branch])
            .current_dir(&self.project_dir)
            .output()?;
        Ok(())
    }
}

pub enum MergeResult {
    Success,
    Conflict(String),
}

    let files_b: HashSet<&str> = b.context.files.iter().map(|s| s.as_str()).collect();
    !files_a.is_disjoint(&files_b)
}
```

```rust
    state: Arc<Mutex<StateStore>>,
    scheduler: Arc<Mutex<Option<Scheduler>>>,
    isolation: Arc<IsolationManager>,
    worker: Arc<dyn Worker>,
}

impl Daemon {
    pub fn new(config: Config, project_dir: &Path) -> Result<Self> {
        let orca_dir = project_dir.join(".orca");
        let state = StateStore::new(&orca_dir)?;
        let worktree_dir = config.worktree_dir(project_dir);
        let codex_config = config.codex_worker_config();

        Ok(Self {
            config: config.clone(),
            project_dir: project_dir.to_path_buf(),
            state: Arc::new(Mutex::new(state)),
            scheduler: Arc::new(Mutex::new(None)),
            isolation: Arc::new(IsolationManager::new(project_dir, &worktree_dir)),
            worker: Arc::new(CodexWorker::new(codex_config)),
        })
    }

    pub async fn run(self) -> Result<()> {
        let socket_path = self.config.socket_path(&self.project_dir);
        info!("starting orca daemon for {:?}", self.project_dir);

        let state = self.state.clone();
        let scheduler = self.scheduler.clone();
        let config = self.config.clone();
        let worker = self.worker.clone();
        let isolation = self.isolation.clone();
        let project_dir = self.project_dir.clone();

        let handler = Arc::new(move |req: RpcRequest| -> RpcResponse {
            let state = state.clone();
            let scheduler = scheduler.clone();
            let config = config.clone();
            let worker = worker.clone();
            let isolation = isolation.clone();
            let project_dir = project_dir.clone();

            // Handle synchronously for now — async handler is a future improvement
            let rt = tokio::runtime::Handle::current();
            rt.block_on(async move {
                handle_request(req, state, scheduler, config, worker, isolation, &project_dir).await
            })
        });

        let server = IpcServer::bind(&socket_path, handler).await?;
        info!("orca daemon ready");
        server.run().await
    }
}

async fn handle_request(
    req: RpcRequest,
    state: Arc<Mutex<StateStore>>,
    scheduler: Arc<Mutex<Option<Scheduler>>>,
    config: Config,
    worker: Arc<dyn Worker>,
    isolation: Arc<IsolationManager>,
    project_dir: &Path,
) -> RpcResponse {
    match req.method.as_str() {
        "orca_plan" => handle_plan(req.id, req.params, state, scheduler).await,
        "orca_status" => handle_status(req.id, req.params, state).await,
        "orca_task_detail" => handle_task_detail(req.id, req.params, state).await,
        "orca_decide" => handle_decide(req.id, req.params, state).await,
        "orca_review" => handle_review(req.id, req.params, state).await,
        "orca_cancel" => handle_cancel(req.id, req.params, state).await,
        "orca_worker_list" => handle_worker_list(req.id, state).await,
        "orca_merge" => handle_merge(req.id, req.params, state, isolation).await,
        "ping" => RpcResponse::success(req.id, serde_json::json!({"pong": true})),
        _ => RpcResponse::error(req.id, crate::protocol::METHOD_NOT_FOUND, &format!("unknown method: {}", req.method)),
    }
}

async fn handle_plan(
    id: Option<serde_json::Value>,
    params: serde_json::Value,
    state: Arc<Mutex<StateStore>>,
    scheduler_lock: Arc<Mutex<Option<Scheduler>>>,
) -> RpcResponse {
    let plan: Plan = match serde_json::from_value(params) {
        Ok(p) => p,
        Err(e) => return RpcResponse::error(id, INVALID_PARAMS, &format!("invalid plan: {}", e)),
    };

    let graph = match DependencyGraph::new(&plan.tasks, &plan.dependencies) {
        Ok(g) => g,
        Err(e) => return RpcResponse::error(id, INVALID_PARAMS, &format!("invalid dependencies: {}", e)),
    };

    let mut state = state.lock().await;
    for spec in &plan.tasks {
        let task = Task::new(spec.clone());
        if let Err(e) = state.add_task(task) {
            return RpcResponse::error(id, INTERNAL_ERROR, &e.to_string());
        }
    }
    state.state_mut().active_plan_id = Some(plan.id.clone());

    let scheduler = Scheduler::new(graph);
    *scheduler_lock.lock().await = Some(scheduler);

    RpcResponse::success(
        id,
        serde_json::json!({
            "plan_id": plan.id,
            "task_count": plan.tasks.len(),
        }),
    )
}

async fn handle_status(
    id: Option<serde_json::Value>,
    params: serde_json::Value,
    state: Arc<Mutex<StateStore>>,
) -> RpcResponse {
    let filter = params
        .get("filter")
        .and_then(|v| v.as_str())
        .unwrap_or("all");

    let state = state.lock().await;
    let tasks: Vec<serde_json::Value> = state
        .all_tasks()
        .iter()
        .filter(|t| match filter {
            "running" => matches!(t.state, TaskState::Running | TaskState::Assigned),
            "blocked" => matches!(t.state, TaskState::Blocked),
            "pending" => matches!(t.state, TaskState::Pending),
            "review" => matches!(t.state, TaskState::Review),
            _ => true,
        })
        .map(|t| {
            serde_json::json!({
                "id": t.spec.id,
                "title": t.spec.title,
                "state": t.state,
                "worker_id": t.worker_id,
            })
        })
        .collect();

    let escalations: Vec<serde_json::Value> = state
        .pending_escalations()
        .iter()
        .map(|e| {
            serde_json::json!({
                "id": e.id,
                "task_id": e.task_id,
                "category": e.category,
                "summary": e.summary,
            })
        })
        .collect();

    RpcResponse::success(
        id,
        serde_json::json!({
            "tasks": tasks,
            "escalations": escalations,
            "active_plan_id": state.state().active_plan_id,
        }),
    )
}

async fn handle_task_detail(
    id: Option<serde_json::Value>,
    params: serde_json::Value,
    state: Arc<Mutex<StateStore>>,
) -> RpcResponse {
    let task_id = match params.get("task_id").and_then(|v| v.as_str()) {
        Some(id) => id,
        None => return RpcResponse::error(id, INVALID_PARAMS, "missing task_id"),
    };

    let state = state.lock().await;
    match state.get_task(task_id) {
        Some(task) => RpcResponse::success(id, serde_json::to_value(task).unwrap()),
        None => RpcResponse::error(id, TASK_NOT_FOUND, &format!("task {} not found", task_id)),
    }
}

async fn handle_decide(
    id: Option<serde_json::Value>,
    params: serde_json::Value,
    state: Arc<Mutex<StateStore>>,
) -> RpcResponse {
    let esc_id = match params.get("escalation_id").and_then(|v| v.as_str()) {
        Some(id) => id.to_string(),
        None => return RpcResponse::error(id, INVALID_PARAMS, "missing escalation_id"),
    };

    let mut state = state.lock().await;
    match state.remove_escalation(&esc_id) {
        Some(esc) => {
            // Resume the blocked task
            if let Some(task) = state.get_task_mut(&esc.task_id) {
                let _ = task.transition_to(TaskState::Running);
                let _ = state.save();
            }
            RpcResponse::success(id, serde_json::json!({"resolved": esc_id}))
        }
        None => RpcResponse::error(
            id,
            crate::protocol::ESCALATION_NOT_FOUND,
            &format!("escalation {} not found", esc_id),
        ),
    }
}

async fn handle_review(
    id: Option<serde_json::Value>,
    params: serde_json::Value,
    state: Arc<Mutex<StateStore>>,
) -> RpcResponse {
    let task_id = match params.get("task_id").and_then(|v| v.as_str()) {
        Some(id) => id.to_string(),
        None => return RpcResponse::error(id, INVALID_PARAMS, "missing task_id"),
    };
    let verdict = match params.get("verdict").and_then(|v| v.as_str()) {
        Some(v) => v.to_string(),
        None => return RpcResponse::error(id, INVALID_PARAMS, "missing verdict"),
    };

    let mut state = state.lock().await;
    match state.get_task_mut(&task_id) {
        Some(task) => {
            let target = match verdict.as_str() {
                "accept" => TaskState::Accepted,
                "reject" => TaskState::Rejected,
                _ => return RpcResponse::error(id, INVALID_PARAMS, "verdict must be 'accept' or 'reject'"),
            };
            match task.transition_to(target) {
                Ok(()) => {
                    let _ = state.save();
                    RpcResponse::success(id, serde_json::json!({"task_id": task_id, "verdict": verdict}))
                }
                Err(e) => RpcResponse::error(id, crate::protocol::INVALID_STATE_TRANSITION, &e.to_string()),
            }
        }
        None => RpcResponse::error(id, TASK_NOT_FOUND, &format!("task {} not found", task_id)),
    }
}

async fn handle_cancel(
    id: Option<serde_json::Value>,
    params: serde_json::Value,
    state: Arc<Mutex<StateStore>>,
) -> RpcResponse {
    let task_id = match params.get("task_id").and_then(|v| v.as_str()) {
        Some(id) => id.to_string(),
        None => return RpcResponse::error(id, INVALID_PARAMS, "missing task_id"),
    };

    let mut state = state.lock().await;
    match state.get_task_mut(&task_id) {
        Some(task) => {
            match task.transition_to(TaskState::Cancelled) {
                Ok(()) => {
                    let _ = state.save();
                    RpcResponse::success(id, serde_json::json!({"cancelled": task_id}))
                }
                Err(e) => RpcResponse::error(id, crate::protocol::INVALID_STATE_TRANSITION, &e.to_string()),
            }
        }
        None => RpcResponse::error(id, TASK_NOT_FOUND, &format!("task {} not found", task_id)),
    }
}

async fn handle_worker_list(
    id: Option<serde_json::Value>,
    state: Arc<Mutex<StateStore>>,
) -> RpcResponse {
    let state = state.lock().await;
    let workers: Vec<serde_json::Value> = state
        .state()
        .workers
        .values()
        .map(|w| serde_json::to_value(w).unwrap())
        .collect();
    RpcResponse::success(id, serde_json::json!({"workers": workers}))
}

async fn handle_merge(
    id: Option<serde_json::Value>,
    params: serde_json::Value,
    state: Arc<Mutex<StateStore>>,
    isolation: Arc<IsolationManager>,
) -> RpcResponse {
    let task_ids: Vec<String> = match serde_json::from_value(
        params
            .get("task_ids")
            .cloned()
            .unwrap_or(serde_json::json!([])),
    ) {
        Ok(ids) => ids,
        Err(e) => return RpcResponse::error(id, INVALID_PARAMS, &format!("invalid task_ids: {}", e)),
    };

    let mut state = state.lock().await;
    let mut results = vec![];

    for task_id in &task_ids {
        if let Some(task) = state.get_task(task_id) {
            if let Some(branch) = &task.branch_name {
                // Attempt merge — actual git operations
                results.push(serde_json::json!({
                    "task_id": task_id,
                    "branch": branch,
                    "status": "merge_requested",
                }));
            }
        }
    }

    RpcResponse::success(id, serde_json::json!({"merges": results}))
}
```

```rust
impl OrcaMcp {
    async fn rpc_call(&self, method: &str, params: serde_json::Value) -> String {
        match IpcClient::connect(&self.socket_path).await {
            Ok(mut client) => {
                let req = RpcRequest::new(method, params);
                match client.call(&req).await {
                    Ok(resp) => {
                        if let Some(err) = resp.error {
                            format!("Error: {}", err.message)
                        } else {
                            serde_json::to_string_pretty(&resp.result).unwrap_or_default()
                        }
                    }
                    Err(e) => format!("RPC call failed: {}", e),
                }
            }
            Err(e) => format!("Cannot connect to orca daemon: {}. Run 'orca daemon start' first.", e),
        }
    }
}

pub async fn run_mcp_server(socket_path: PathBuf) -> anyhow::Result<()> {
    let server = OrcaMcp::new(socket_path);
    let transport = rmcp::transport::io::stdio();
    let service = server.serve(transport).await?;
    service.waiting().await?;
    Ok(())
}
```

# Orca

Multi-agent orchestrator: Claude Code brain + Codex workers.

Orca lets Claude Code (CC) act as the brain — planning, reviewing, and making decisions — while dispatching implementation tasks to Codex workers running in parallel. A lightweight daemon coordinates everything through structured messaging, not terminal buffer hacks.

| Problem | Orca's approach |

| No CC → Codex orchestration | CC as brain, Codex as workers, daemon as middle layer |
